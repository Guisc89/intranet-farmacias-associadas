# intranet-farmacias-associadas

Protótipo de intranet desenvolvido para otimizar a comunicação, gestão e suporte aos associados da Rede **Farmácias Associadas**.  
O projeto foi realizado e desenhado integralmente no **Figma**, com foco em **usabilidade**, **organização de informações** e **entrega ágil** em parceria com a equipe de desenvolvimento contratada.

## 🎯 Objetivo
Criar um ambiente centralizado para os associados acessarem:
- Materiais de marketing prontos para download
- Solicitação de materiais personalizados
- Calendário de campanhas e eventos
- Informações comerciais e listas de fornecedores
- Notícias e comunicados internos
- Atalhos rápidos para serviços essenciais

## 💼 Meu papel no projeto
- **Design UX/UI**: Criação do layout, fluxos e hierarquia visual no Figma.
- **Liderança de projeto**: Acompanhamento das sprints, facilitando comunicação entre stakeholders e desenvolvedores.
- **Validação ágil**: Coleta de feedbacks rápidos e ajustes contínuos para garantir aderência às necessidades reais dos usuários.

## 📌 Funcionalidades principais
- **Dashboard** com destaques, notícias e próximos eventos
- **Módulo Marketing**: solicitações, downloads e kits
- **Módulo Comercial**: listas, fornecedores e condições comerciais
- **Calendário integrado** de campanhas e treinamentos
- **Seção de Atalhos** para funcionalidades mais usadas
- **Formulários inteligentes** para pedidos e envios de arquivos

## 🛠️ Ferramentas e tecnologias utilizadas
- **Figma** (protótipo e design system)
- **Photoshop** (edição e tratamento de imagem)
- **Metodologias ágeis**: acompanhamento em sprints e feedback loops
- **(Futuro desenvolvimento)**: HTML5, CSS3, JavaScript e framework frontend (React ou Vue)

## 🚀 Os resultados esperados
- Redução no tempo de solicitação e acesso a materiais
- Maior clareza na comunicação interna
- Otimização no fluxo de trabalho entre associados e matriz
- Interface intuitiva para uso diário
